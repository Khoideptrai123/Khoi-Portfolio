Virtual Oscillscope using LabView

fall '20

technical skills: LabView, Signal
soft skills: iterative design review, team coordination

An oscilloscope is one of the most useful diagnostic tools in the design, and maintenance of electronic, and electromechanical systems. In this project, I have built a virtual oscilloscope using LabView with a few basic time and frequency domain features.

The scope features reading from either .csv, .wav or even direct audio input from the PC. Then it will display relevant time-domain features of the signal such as amplitude, RMS value, frequency. It also provided horizontal/vertical cursors for measuring different points.

Beside, the scope also have Frequency Domain feature by showing Fourier Fast Transfrom (FFT) of the signal. Then 3 filters are added for users to control: Low Pass Filter, High Pass Filter and Moving Average Filter
